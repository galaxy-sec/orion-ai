# 函数工具文档

**Git操作:**
- git-status   - 仓库状态查询
- git-add      - 添加文件到暂存区
- git-commit   - 提交更改
- git-push     - 推送提交
- git-pull     - 拉取更新
- git-diff     - 查看差异
- git-log      - 查看提交历史

**文件系统:**
- fs-ls   - 列出目录
- fs-cat  - 查看文件内容
- fs-find - 搜索文件
- fs-pwd  - 当前目录
- fs-cd   - 更改目录

**系统信息:**
- sys-uname - 系统信息
- sys-ps    - 进程列表
- sys-df    - 磁盘空间
- net-ping  - 网络连通性测试
